Sobre o projeto

Objetivo

Este projeto foi desenvolvido por Felipe Melo, Poliana Campelo e João Neto, alunos do Mestrado Profissional em Tecnologia da Informação do Instituto Federal da Paraíba. O projeto foi idealizado para fazer parte do projeto final da disciplina de Banco de Dados, tendo como objetivo, apresentar informações, através de gráficos e visualizações, sobre a o avanço da vacinação para a Covid-19 no estado da Paraíba, utilizando, para isso, fonte de dados abertas e consequentemente, auxiliar a população na obtenção e entendimento dessas informações.

Visualizações dos dados

Visualizações são representações gráficas de informações obtidas a partir de dados. As visualizações apresentadas neste projeto foram criadas com o auxílio da ferramenta MongoDB Charts, utilizando para isto, um conjunto de dados previamente processados e carregados utilizando o MongoDB como ferramenta de banco de dados. As visualizações foram incorporadas neste site a partir de opções de Embedding , oferecidas pela ferramenta.

Bases de dados

As bases de dados utilizadas para a criação do banco de dados e consequetemente das visualizações do projeto são duas bases de dados abertas.

A primeira é a base de dados da campanha nacional de vacinação, disponibilizada pelo Datasus, a partir do link: Base de dados Datasus - Covid-19 vacinação

A segunda é a base de dados do IBGE, contendo informações sobre a população estimada de cada município, disponibilizada pelo IBGE, a partir do link: Base de dados IBGE - População estimada
